English接客用 注文マニュアル

1. お客様を迎える

英語: Hi there! / Welcome!

ハイ ゼア! / ウェルカム!

2. 注文を聞く

英語: What would you like to order? / What can I get for you?

ワット ウッジュー ライク トゥ オーダー? / ワット キャナイ ゲット フォー ユー?

3. 注文内容を確認

英語: Is that everything? / Is that all? / Is that all for your order?

イズ ザット エヴリシング? / イズ ザット オール? / イズ ザット オール フォー ユア オーダー?

4. セットメニューの提案

英語: Would you like to add fries to that? / Would you like to make it a combo with fries?

ウッジュー ライク トゥ アッド フライズ トゥ ザット? / ウッジュー ライク トゥ メイク イット ア コンボ ウィズ フライズ?

5. お会計を確認

英語: It’s 3000 yen.

イッツ スリーサウザンド イェン.

支払い方法: Will you be paying with cash? / We accept PayPay or cash.

ウィル ユー ビー ペイング ウィズ キャッシュ? / ウィ アクセプト ペイペイ オア キャッシュ.

6. 注文が出来上がったら

英語: Excuse me, your order is ready! / The person who ordered the burger!

エクスキューズ ミー, ユア オーダー イズ レディ! / ザ パーソン フー オーダード ザ バーガー!

7. 商品を渡す

英語: Here you go. Thank you so much.

ヒア ユー ゴー. サンキュー ソー マッチ.

8. アレルギーやヴィーガン対応

英語: Do you have any allergies? / Is this vegan? / Does this contain any nuts or dairy?

ドゥー ユー ハヴ エニ アレルジーズ? / イズ ディス ヴィーガン? / ダズ ディス コンテイン エニ ナッツ オア デイリー?
